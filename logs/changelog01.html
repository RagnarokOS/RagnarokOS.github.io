revision: 1.15
date: 2024-09-29

* ragnarok-installer: now using a dedicated, unprivileged user named
  '_installer' for functions that fetch files off the internet instead
  of using the default live user.
* ragnarok-base: the security-check cronjob now only verifies bootloader
  files if they are present (and syslinux.cfg was added).
* oKSH: merged 'Enforce no invalid characters in shell scripts' from
  OpenBSD.
* Added chrony to keep system time in sync.
