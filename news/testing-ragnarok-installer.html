The Ragnarok Installer Now Available for Testing

The latest live ISO[1] includes the first version of Ragnarok's installer package. Anyone interested in testing it may do so and can use its github repo's issue tracker[2] to report any bug, if found.

It is advised to test the installer in a virtual machine or spare computer before trying it on a system of any importance. If guidance is needed, you may refer to the official installation guide[3].
