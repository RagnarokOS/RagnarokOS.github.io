Ragnarok Switching to a Gentoo Base

For the past few weeks, work has been underway to re-base Ragnarok on Gentoo. This new version will offer a binary base, with "third party packages" being compilable from source. Binary packages will be provided for some common packages, however.

About the current Debian version

The current, Debian based version of Ragnarok can still be used, at least until Debian Bookworm becomes EOL. People currently using it can keep doing so, but those using Ragnarok's kernel build will need to make sure to switch to Debian's kernel, as there will be no more update to Ragnarok's build.

That said, it will be highly recommended to switch to the new version as soon as it is released.
