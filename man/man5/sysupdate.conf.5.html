NAME

sysupdate.conf — sysupdate configuration file

DESCRIPTION

The /etc/sysupdate.conf file contains the configuration options for the sysupdate(8) command.

It follows the OPTION="value" format. Any line that doesn't follow this format is ignored.

OPTIONS

The following options are supported

MIRROR
The mirror used to fetch updates from.
VERSION
The current version of the installed system.
PUBKEY
Path to Ragnarok's public signify key.
